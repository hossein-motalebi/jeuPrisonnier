# Jeu du Dilemme du Prisonnier

Ce projet est implémenté en tant que monorepo, ce qui signifie que tout le code source est contenu dans un seul répertoire. Le backend est construit avec Spring Boot et fonctionne sur le port 8080, tandis que le frontend est construit avec Angular et fonctionne sur le port 4200.

## Comment Jouer

Pour jouer au jeu, suivez ces étapes :

1. Connectez-vous au VPN de l'université car le jeu est déployé sur le eCloud de l'université.
2. Ouvrez votre navigateur et accédez à : [http://129.88.210.103:4200/]

Bonne chance !
